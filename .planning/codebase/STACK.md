## Configuration

**Environment Variables:**
- Stored in `.env.local` (not committed)
- `.env.example` serves as template (present)

**Required env vars:**
- `AUTH_SECRET` - NextAuth JWT secret (generated via `openssl rand -base64 32`)
- `AUTH_GOOGLE_ID` - Google OAuth client ID (triggers Google provider)
- `AUTH_GOOGLE_SECRET` - Google OAuth client secret (triggers Google provider)
- `ALLOWED_EMAILS` - Comma-separated list of allowed email addresses (empty = all emails allowed)
- `GOOGLE_SHEETS_ID` - Spreadsheet ID for Google Sheets API (optional; mock data used if absent)
- `GOOGLE_SERVICE_ACCOUNT_EMAIL` - Service account email (optional)
- `GOOGLE_PRIVATE_KEY` - Private key JSON (optional; `\\n` replaced with actual newlines in code)

**Build Configuration:**
- `next.config.ts` - Next.js config (remote image patterns for Google profile pictures)
- `tsconfig.json` - TypeScript config (ES2017 target, strict mode, path aliases `@/*`)
- `eslint.config.mjs` - ESLint configuration (Next.js web vitals + TypeScript rules)
