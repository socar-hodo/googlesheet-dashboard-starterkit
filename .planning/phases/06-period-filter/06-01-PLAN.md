    구현 순서:
    1. PeriodKey 타입 및 DateRange 인터페이스 정의
    2. DAILY_PERIODS, WEEKLY_PERIODS, PERIOD_LABELS, DEFAULT_DAILY_PERIOD, DEFAULT_WEEKLY_PERIOD 상수
    3. toISODate(d: Date): string — d.toISOString().slice(0, 10)
    4. getMonday(d: Date): Date — JS getDay() 기반 ISO 주(월요일 시작) 계산
       - day = d.getDay() (0=일, 1=월, ..., 6=토)
       - diff = day === 0 ? -6 : 1 - day
       - 새 Date에 setDate 후 setHours(0,0,0,0)
    5. getDateRange(period: PeriodKey, today: Date = new Date()): DateRange
       - this-week: getMonday(today) ~ today
       - last-week: getMonday(today)-7일 ~ getMonday(today)-1일
       - this-month: new Date(year, month, 1) ~ today
       - last-month: new Date(year, month-1, 1) ~ new Date(year, month, 0)
       - 주의: JS Date는 음수 month 자동 처리 → new Date(2026, -1, 1) = 2025-12-01 (올바름)
    6. filterDailyByPeriod(records: DailyRecord[], range: DateRange): DailyRecord[]
       - r.date >= range.start && r.date <= range.end (문자열 사전순 = 날짜 순서)
    7. parseWeekMonth(week: string): number | null
       - /^(\d+)월/.exec(week) → parseInt(match[1])
       - 매칭 없으면 null
    8. filterWeeklyByPeriod(records: WeeklyRecord[], period: 'this-month' | 'last-month', today: Date = new Date()): WeeklyRecord[]
       - currentMonth = today.getMonth() + 1 (1-indexed)
       - lastMonth = currentMonth === 1 ? 12 : currentMonth - 1
       - targetMonth = period === 'this-month' ? currentMonth : lastMonth
       - parseWeekMonth 결과 null이면 전체 포함 (폴백), 아니면 month === targetMonth 인 것만
