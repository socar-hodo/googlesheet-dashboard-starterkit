  <name>Task 1: xlsx CDN 설치</name>
  <files>package.json</files>
  <action>
SheetJS 최신 버전을 CDN tarball URL로 설치한다.

중요: `npm install xlsx`는 절대 사용 금지 — npm 레지스트리 버전은 0.18.5로 보안 취약점(Denial of Service, Prototype Pollution) 포함.
반드시 CDN URL 설치:

```bash
npm install https://cdn.sheetjs.com/xlsx-0.20.3/xlsx-0.20.3.tgz
```

설치 후 `package.json`의 `dependencies`에 다음과 유사한 항목이 추가되어야 한다:
```json
"xlsx": "https://cdn.sheetjs.com/xlsx-0.20.3/xlsx-0.20.3.tgz"
```

TypeScript 타입: CDN tarball에 내장 타입 포함. `import { utils, writeFile } from 'xlsx'`가 타입 오류 없이 동작하면 별도 `@types/xlsx` 불필요.

// 숫자값은 포맷 없는 raw number — 데이터 처리 용이 (₩ 포맷 사용 금지)
