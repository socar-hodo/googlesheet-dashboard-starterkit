# Phase 4: Data Table + Polish - Context

**Gathered:** 2026-02-24
**Status:** Ready for planning

<domain>
## Phase Boundary

Daily/Weekly 탭에 일별·주별 상세 데이터 테이블을 추가하고, 합계/평균 요약 행과 마지막 데이터 업데이트 타임스탬프를 표시한다. 새로운 탭 구조나 필터링/검색 기능은 이 단계 범위 밖이다.

</domain>

<decisions>
## Implementation Decisions

### 테이블 시각 스타일
- Striped 행: 짝수/홀수 행 배경색 교체로 가독성 확보
- 헤더 고정(sticky): 스크롤 시 컬럼 제목 항상 표시
- 정렬 기능 없음: 날짜 범위는 고정되어 있어 정렬 필요 없음
- 배치: 차트 아래에 테이블 직접 배치 (Daily/Weekly 탭 각각 동일 패턴)

### 요약 행 설계
- 합계 행과 평균 행 둘 다 표시 (테이블 하단 2행)
- 굵은 폰트 + 진한 배경으로 일반 데이터 행과 시각적으로 명확히 구분
- 이용건수 요약: 합계 (총 이용건수)
- 이용시간 요약: 합계 (총 이용시간)
- 가동률 요약: 평균 가동률 (합계는 의미 없음)
- 손익 요약: 합계 + 평균 둘 다

### 셀 데이터 포맷
- 매출/손익: 원 단위 전체 (₩1,234,567 형식) — KPI 카드의 만원 단위와 별도로 상세 테이블에서는 정확한 금액 표시
- 손익 음수(적자): 빨간색 텍스트로 즉시 식별 가능하게 표시
- 이용시간: "Xh Ym" 형식 (예: 2h 30m) — 직관적 읽기
- 가동률: 소수점 한 자리 % (예: 75.3%) — 정밀도 중시

### 업데이트 타임스탬프
- 위치: 대시보드 헤더 영역 우측
- 형식: 상대 시간 + 절대 시간 동시 표시 (예: "3시간 전 (2026-02-24 14:30)")
- 업데이트 시점: 페이지 로드 시 1회 읽기 (자동 새로고침 없음)
- 데이터 소스: 스프레드시트의 마지막 데이터 행 날짜를 타임스탬프로 사용

### Claude's Discretion
- 테이블 행 높이 및 패딩 값
- Striped 행의 정확한 배경 색조 (기존 shadcn/ui 테마 변수 활용)
- "합계" / "평균" 레이블 텍스트 및 셀 정렬
- 로딩 스켈레톤 디자인

</decisions>

- 타임스탬프는 "3시간 전 (2026-02-24 14:30)" 패턴으로 간결함과 정확성 동시 제공
